# GDP Term 1 Course Guide

This repository contains the current Term 1 materials for a computer science fundamentals track. The lessons already written focus on core CS foundations, Python, algorithmic thinking, data structures, and algorithms.

## Course Structure

### 1. Introduction to Computer Science

Focus: what computer science is, where computing came from, and why algorithms matter.

- [What is Computer Science?](./cs_fundamentals_course/01_Introduction_to_Computer_Science/01_What_is_Computer_Science.md)
- [History of Computing](./cs_fundamentals_course/01_Introduction_to_Computer_Science/02_History_of_Computing.md)
- [What is an Algorithm?](./cs_fundamentals_course/01_Introduction_to_Computer_Science/03_What_is_an_Algorithm.md)

### 2. Programming Fundamentals

Focus: a beginner-friendly introduction to Python, why it is useful, and how to set up a working environment.

- [Introduction to Python](./cs_fundamentals_course/02_Programming_Fundamentals/01_Introduction_to_Python.md)

### 3. Algorithmic Thinking

Focus: breaking problems into steps, recognizing patterns, hiding unnecessary complexity, designing solutions, and analyzing efficiency.

- [Introduction to Algorithmic Thinking](./cs_fundamentals_course/03_Algorithmic_Thinking/01_Introduction_to_Algorithmic_Thinking.md)
- [Decomposition](./cs_fundamentals_course/03_Algorithmic_Thinking/02_Decomposition.md)
- [Pattern Recognition](./cs_fundamentals_course/03_Algorithmic_Thinking/03_Pattern_Recognition.md)
- [Abstraction](./cs_fundamentals_course/03_Algorithmic_Thinking/04_Abstraction.md)
- [Algorithm Design](./cs_fundamentals_course/03_Algorithmic_Thinking/05_Algorithm_Design.md)

#### Big O Notation

Focus: time complexity, space complexity, best/average/worst case analysis, and practice exercises.

- [Big O Notation Lesson](./cs_fundamentals_course/03_Algorithmic_Thinking/06_Big_O_Notation/01_Big_O_Notation_Lesson.md)
- [Big O Notation Exercises](./cs_fundamentals_course/03_Algorithmic_Thinking/06_Big_O_Notation/03_Big_O_Notation_Exercises.md)
- [Big O Notation Complete Notes](./cs_fundamentals_course/03_Algorithmic_Thinking/06_Big_O_Notation/Big_O_Notation_Complete.md)
- [Big O Notation PDF](./cs_fundamentals_course/03_Algorithmic_Thinking/06_Big_O_Notation/Big_O_Notation_Complete.pdf)

#### Time and Space Complexity

Focus: the specific study of runtime growth, memory usage, and how complexity connects back to algorithm efficiency.

- [Time and Space Complexity](./cs_fundamentals_course/03_Algorithmic_Thinking/07_Time_and_Space_Complexity/01_Time_and_Space_Complexity.md)

### 4. Data Structures

Focus: how data is organized in memory, how core structures behave, and when to use each one.

- [Beginner Data Structures: The Complete Guide](./cs_fundamentals_course/04_Data_Structures/01_Beginner_Data_Structures.md)
- [Advanced Data Structures](./cs_fundamentals_course/04_Data_Structures/02_Advanced_Data_Structures.md)
- [Lesson 1: Arrays](./cs_fundamentals_course/04_Data_Structures/Arrays.md)
- [Lesson 2: Linked Lists](./cs_fundamentals_course/04_Data_Structures/Linked_List.md)

Topics covered in the beginner guide:

- Arrays
- Strings
- Linked Lists
- Stacks
- Queues
- Hash Tables
- Trees
- Graphs

Topics highlighted in the advanced guide:

- AVL Trees
- Red-Black Trees
- Min-Heaps and Max-Heaps
- Tries
- Directed, Undirected, and Weighted Graphs

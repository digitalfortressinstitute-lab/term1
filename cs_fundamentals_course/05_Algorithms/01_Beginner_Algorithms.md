# Beginner Algorithms

This is a list of fundamental algorithms ideal for beginners.

### Sorting Algorithms

These are great for learning about complexity and how to compare different approaches to solving a problem.

1.  **Bubble Sort:** The simplest sorting algorithm to understand and implement.
2.  **Insertion Sort:** Another simple sorting algorithm that is efficient for small lists.
3.  **Selection Sort:** A simple sorting algorithm that is good for understanding the concept of "in-place" sorting.
4.  **Merge Sort:** A very efficient and stable sorting algorithm. It's a great introduction to the "divide and conquer" strategy.
5.  **Quick Sort:** Generally faster than Merge Sort in practice, but not stable. Also a "divide and conquer" algorithm.

### Searching Algorithms

These are fundamental for any programmer.

1.  **Linear Search:** The most basic way to search for an item in a list.
2.  **Binary Search:** A much more efficient searching algorithm, but it requires the list to be sorted.

### Graph Traversal

Once you understand basic graphs, these are the first algorithms to learn.

1.  **Breadth-First Search (BFS):** A fundamental graph traversal algorithm.
2.  **Depth-First Search (DFS):** Another fundamental graph traversal algorithm.

### String Matching

1.  **Naive String Matching:** The simplest method to solve the string matching problem.
